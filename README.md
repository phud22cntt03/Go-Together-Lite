## Firebase Secrets

For local development, copy [`.firebase.secrets.json.example`](.firebase.secrets.json) to `.firebase.secrets.json` and fill in your Firebase values.

Run the app with:

```powershell
./scripts/run_with_firebase_secrets.ps1
```
